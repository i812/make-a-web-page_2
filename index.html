Summary of Concepts

HTML Basics

The web consist of a bunch of computers sharing documents over the internet. A typical web page is stored on a server and when a client computer wants to use that document it uses a web browser to read the HTML documents stored on the server. The web browser decodes the document on to the user's computer screen.

  • HTML - Hypertext Markup Language - Makes up a huge portion of the web's content.
  • Contains the text that makes up the majority of all of the web pages on the World Wide Web.
  • These HTML documents also contain tags and elements that are used to format how the document looks.
  • HTML elements can be either inline or block
  • Inline - tend to flow along within the text content
  • Block - tend to expand naturally to fill their parent container and can have margins and padding.

Making a Structured HTML Document

There are a lot of tools and techniques that the web developer can leverage to make the construction of a good web page easier. Combine those with a good understanding of how to break down and manipulate HTML and you can build a nice looking page pretty easily.

  • Tree Structure - HTML elements can contain other elements inside of them
  • Similar to a parent to child relationship
  • Traits from one level are passed on to the lower ones
  • Developer Tools - Where do we find them and what are they useful for
  • Can be found in the web browser
  • Can be used to "play" with existing web pages to learn about how they are formatted
  • Choosing a Text Editor - What to use when writing HTML
  • Programmers need special editors - Such as Notepad++ or Sublime Text 2
  • Many have auto formatting and auto completion for standard commands and tags
